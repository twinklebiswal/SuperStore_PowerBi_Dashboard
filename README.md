# SuperStore_PowerBi_Dashboard
This Project is an interactive sales
## Tools Used
-Microsoft Power Bi
-Excel
## Dashboard Features 
-Total Sales , Profit , Discount and Quantity overview
-Profit analysis by Region
-Sales analysis by category
-Profit by Ship Mode
-Sales distribution by Segment
-City level analysis
-Interactive State filter
## Key Insights
-West region generates the highest Profit.
-Technology category has the highest Sales.
-Standard shipping mode contributes the most Profit.
